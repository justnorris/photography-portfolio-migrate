# Easy Photography Portfolio: Migrate

**PHP 5.4+ Required**

You can use this plugin to migrate portfolio posts and galleries to Easy Photography Portfolio. This plugin is built to migrate [Colormelon WordPress Themes](https://colormelon.com/themes) to [Easy Photography Portfolio](http://easyphotographyportfolio.com) plugin format.

**Note**: some in-code configuration is going to be required.

If you want to make this work with your theme/plugin, pay attention to the `phormig_settings` filter in [core.php](core.php).
